# 中国建材集团有限公司AI市集项目采购需求与要求文档

## 一、建设背景

人工智能技术已成为推动中国建材集团业务创新与生产力重塑的关键引擎。随着集团数字化战略持续推进，内部智能化应用呈现爆发式增长，各类智能体在辅助决策、业务流程自动化等核心业务场景中逐步落地应用。

当前，集团智能化建设长期受烟囱式发展模式制约，面临资产管控无序与研发效能低下的双重发展困境。一方面，集团缺乏智能化资产统一治理体系，大量优质智能体资源分散存储、运行于各成员企业及部门内部，形成严重的应用孤岛。因无集团级统一纳管机制，智能体资产底数不清、运行状态不明，无法实现集团层面的分级统筹与优化配置，资产利用率极低。另一方面，资源孤岛问题直接导致重复建设乱象，跨组织、跨层级的AI资源共享与协作机制缺失，通用型智能体能力无法在集团内部流通复用。各下属单位长期开展低水平重复开发，造成算力资源、项目资金、研发人力的严重浪费，无法形成规模化智能赋能效应，大幅拉低了集团数字化、智能化建设的整体投入产出比。

为彻底打破业务与技术壁垒，盘活集团存量AI资产，构建统一、共享、可控的智能化资源生态，亟需建设集团级AI市集平台，实现AI资产的统一汇聚、规范管控、高效复用与持续迭代。

## 二、项目目标

本项目聚焦破解集团智能化建设资产碎片化、建设重复化核心痛点，立足集团整体数字化发展布局，打造集资产聚合、分级管控、资源共享、能力进化于一体的集团级AI市集平台。通过搭建统一智能化数智底座，实现全集团AI资产有序流转、高效复用与价值深挖，重塑集团智能化治理体系，全面释放AI技术规模化赋能价值，具体核心目标如下：

1. **构建统一汇聚平台**：搭建集成智能体发布、发现、调用、评价、迭代全流程能力的集团级统一AI市集平台，彻底打通内部资源壁垒，实现AI资产全生命周期标准化管理。

2. **落实集约化管控体系**：建立全集团统一的智能体准入标准、认证体系与管理规范，实现全集团智能化应用统一纳管、实时监控、动态管控，全面摸清集团AI资产底数与运行状态。

3. **实现降本增效提质**：搭建跨层级、跨单位的AI资源共享机制，推动通用型智能体能力全域流通复用，杜绝下属单位重复开发建设，大幅降低智能化建设成本，提升集团整体智能化投入产出比。

4. **沉淀高价值数字资产**：建立市场化AI资产评价迭代机制，筛选、固化适配建材行业业务场景的精品智能体，将企业隐性业务经验、专家知识转化为可复用、可沉淀的显性数字资产，构建集团专属行业知识壁垒。

5. **搭建多层级管控体系**：打造纵向穿透式资产可视与管控机制，支持上级单位追溯全量AI资源、向下赋能基层业务，同时保障各级单位管理员自主管控、独立审批权限，实现分级负责、全域可控。

## 三、项目核心建设内容

本项目需建成一套完整、成熟、安全、可扩展的集团级AI市集系统，作为集团统一的智能化基础设施，全面支撑智能体发布、发现、调用、评价、迭代等核心业务功能，具体建设内容如下：

### （一）总体平台架构建设

平台采用分层微服务架构设计，自上而下分为用户展现层、统一网关层、核心业务层、数据服务层、基础设施层，各层级独立解耦、协同联动，保障平台高并发、高可用、高安全运行。

1. **用户展现层**：搭建Web端、企业微信H5双端统一交互门户，实现一端多触点适配PC端与移动端使用场景；深度对接集团统一身份认证系统，实现用户无感免登、全域通行。

2. **统一网关层**：部署云原生API网关，支持万级QPS高并发承载能力；实现基于API Key的零信任安全鉴权、自适应流控、熔断降级机制，同时完成Token消耗统计、调用日志异步采集与计量分析。

3. **核心业务层**：基于微服务架构拆解搭建独立中台模块，包含资产管理中心、多租户管理中心、门户展示中心、组织架构服务、运营反馈中心，支撑平台全业务流程独立、高效运行。

4. **数据服务层**：构建结构化、时序、对象混合存储底座，适配不同类型数据存储需求。采用关系型数据库存储强一致性业务数据；通过时序数据库实现海量计量日志高吞吐写入；依托对象存储完成智能体配置快照、知识库切片的持久化存储。

5. **基础设施层**：基于容器化集群搭建统一算力底座，实现计算资源池化管理、动态调度与弹性伸缩，适配平台业务峰值负载与长期扩容需求。

### （二）核心业务模块建设

1. **多维资产发现与交互模块**：支持用户跨组织、跨层级穿透查看已授权智能体资源，提供热度排序、用户评分、业务标签等多维检索能力，支持智能体在线即时试用，快速验证业务适配性。

2. **白盒化洞察与敏捷复用模块**：开放智能体核心编排逻辑可视化展示，支持智能体资产配置导出、一键克隆副本、二次开发迭代，形成“资源引入\-技术吸收\-创新迭代”的业务闭环，提升资产复用率。

3. **自主上架申报模块**：搭建标准化、流程化的智能体自主上架通道，支持普通用户将自研优质智能体提交本级管理员审核，审核通过后在授权范围内上线使用，激活基层创新能力。

4. **分级准入与全局管控模块**：建立严格的智能体上架审核防火墙，实行本级初审、跨级提级复核双重审核机制；赋予管理员对违规、闲置、高风险智能体一键强制下架、冻结管控权限。

5. **资产提级与优胜劣汰模块**：搭建优质AI资产主动发掘与晋升通道，支持将下级单位高潜力智能体提级为集团公共共享资产；依托用户真实评分、平台活跃度等数据，建立智能体末位清洗机制，保障平台资产质量。

6. **多维度实时运营看板模块**：为集团、各级子公司管理员提供定制化运营看板，实时展示智能体调用次数、算力消耗分布、覆盖企业范围、用户评价等核心运营数据，支撑精细化运营决策。

### （三）功能模块详细需求

系统按用户角色划分为普通用户端、管理员端两大功能体系，覆盖全场景业务需求。

#### 1\. 普通用户端功能

1. **多维资产发现与检索功能**：构建基于集团组织架构的层级化智能资产视图，支持用户跨层级穿透查询本级、上级已合规上线的智能体；支持通过热度、评分、业务场景标签、关键词等多维度筛选检索，快速精准定位所需AI能力资源。

2. **交互式试用与白盒化洞察功能**：支持所有上线智能体在线免费试用，直观验证业务适配效果；开放智能体编排逻辑、流程细节白盒展示，帮助用户深度理解技术原理与应用场景；支持用户对使用过的智能体进行评分、评论反馈，构建完整的资产评价闭环。

3. **自主贡献与上架申报功能**：开放基层创新入口，支持员工将个人、团队自研的优质智能体提交本级平台，自动触发标准化审核流程，经管理员审核通过后，可在对应权限范围内公开使用，打通基层创新成果转化通道。

#### 2\. 管理员端功能

1. **分级准入与合规审核中心**：负责本级用户智能体上架申请全流程审核，同步复核下级单位智能体提级发布申请；重点审核智能体政治合规性、数据安全性、业务适用性，严格把控平台资产准入质量。

2. **资产价值发现与提级运作功能**：依托平台运营数据，主动挖掘下级单位高使用率、高评分、高适配性的优质智能体，支持将局部优质资产提级为本级或集团全域公共资产，最大化盘活AI资产价值。

3. **穿透式全局管控与应急熔断功能**：支持管理员对所辖范围内各级单位管理员权限进行配置、调整、收回，实现全域权限管控；针对违规上线、存在安全风险、长期闲置的智能体，可一键下架、冻结，快速阻断风险扩散，保障平台生态安全合规。

4. **运营监控与用量看板功能**：专属各级单位资产管理后台，实时追踪智能体审核状态、上线版本、运行时长；动态监控智能体调用次数、Token消耗量、覆盖用户及企业数量、用户评分等核心数据，为资源配额调整、资产优胜劣汰、业务优化迭代提供数据支撑。

### （四）关键管控技术与安全机制需求

1. **多租户与数据主权隔离技术**：采用“运营数据硬隔离、资产数据软连通”混合架构设计，在保障集团资产共享复用的前提下，严格杜绝跨企业敏感数据越权访问，全面保障各成员企业数据主权独立、数据安全可控。

2. **纵向穿透式权限与组织映射技术**：与集团主数据平台实时同步组织架构信息，建立上级监管下级、逐级穿透的递归可见视图与权限管控体系，实现组织架构、权限体系动态匹配、精准管控。

3. **精细化计量与预算配额流控技术**：搭建原子化异步计量引擎，从企业、应用双维度精准统计资源用量；配置基于预算的动态配额策略，流量、用量触及阈值时自动实现毫秒级阻断，避免资源滥用、超额占用。

4. **立体化运维与智能监控告警机制**：搭建全链路可观测运维平台，实时监测系统QPS、服务器负载、接口响应等核心指标；配置异常流量、故障场景自适应降级策略，集成企业微信、短信分级告警机制，实现故障早发现、早预警、早处置。
